# 🚀 Featured Projects

## 🤸 Prog Ginnastica
> Personalized rehabilitation and gymnastics training platform powered by collaborative filtering.

A full-stack web application that recommends exercises based on injury history, training profile and user feedback using a KNN recommendation engine.

### Tech Stack
- React + Vite + TailwindCSS
- Flask REST API
- scikit-learn
- pandas / NumPy

### Highlights
- Collaborative filtering recommendation system
- Automatic retraining based on user feedback
- Interactive dashboard and training tracking
- Client-side fallback recommender system

## 🍳 Gordon RamsAI
> AI culinary assistant built for the course *LLM & Generative AI*.

University project focused on two approaches:
1. Fine-tuning `T5-small` for recipe summarization
2. Ollama pipeline with prompting strategies exposed through Flask APIs

### Tech Stack
- Python
- HuggingFace Transformers
- Ollama
- Flask
- React

### Highlights
- Fine-tuning experiments with ROUGE evaluation
- Chain-of-Thought and Few-Shot prompting
- Custom AI chef persona
- Local LLM pipeline integration
